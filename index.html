Instalación Paso a Paso

Sigue estos pasos para instalar el generador correctamente.

1
Descarga un navegador compatible
Descarga e instala un navegador compatible. Para Android se recomienda Lemur Browser. En PC cualquier navegador basado en Chromium.
2
Descarga el Generador
Presiona el botón OBTENER GENERADOR de esta página para descargar el archivo .zip del generador.
3
Abre la página de extensiones
En tu navegador escribe chrome://extensions en la barra de direcciones y presiona Enter.
4
Activa el Modo Desarrollador
Activa el interruptor Developer mode que se encuentra en la esquina superior derecha.
5
Carga el Generador
Presiona Load *.zip y selecciona el archivo descargado. ¿No aparece ese botón? Arrastra el archivo .zip directamente a la página y acepta.
6
¡Listo! La imagen se generará automáticamente
Una vez instalado, navega a la página del perfil de Roblox y la extensión generará y guardará la imagen de forma automática.
